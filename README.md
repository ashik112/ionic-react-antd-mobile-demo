# IONIC REACT + ANTD MOBILE
Integrate antd mobile with ionic react

## Install
```bash
npm install
```
## Run
```bash
ionic serve
```
## Documentation

[Ant Design Mobile](https://mobile.ant.design/docs/react/introduce) & 
[Ionic React](https://ionicframework.com/docs/react/your-first-app)
